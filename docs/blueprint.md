# **App Name**: GeoExplorer

## Core Features:

- Map Display: Display an interactive map using Leaflet.
- JSON File Upload: Allow users to upload a .json file containing location data (name, id_dane, latitude, longitude).
- Location Marker Display: Display markers on the map for each location in the .json file.
- Department Dropdown: Create a dropdown list of departments from the .json data.
- Municipality Dropdown: Create a dropdown list of municipalities from the .json data, filtered by the selected department.
- Location Filtering: Filter the location markers displayed on the map based on the selected department and municipality.

## Style Guidelines:

- Primary color: Sky blue (#87CEEB) to evoke a sense of exploration and openness.
- Background color: Light gray (#F0F0F0) for a clean and neutral backdrop.
- Accent color: Teal (#008080) to highlight interactive elements and markers.
- Body and headline font: 'PT Sans' (sans-serif) for readability and a modern feel.
- Use clear and recognizable map icons to represent location data.
- The map should take up the majority of the screen, with dropdown filters positioned above or to the side.
- Use subtle zoom animations on markers when the location is selected.
